## 6. Recommandations d'ajustement

### 6.A — `criteria.md` (ajustements structurels proposés)

**Proposition #1 — Élargir F1 single en période playoffs majeurs**
```diff
- Single : 1.50 – 2.00 (sweet spot 1.65 – 1.90)
+ Single : 1.50 – 2.00 (defaut)
+ Single PLAYOFF MODE : 1.50 – 2.50 si conditions remplies (cf F1-bis)
```
Avec une nouvelle clause :
```
### F1-bis — Playoff exception (mai-juin NBA/NHL)
Pendant les Conference Finals et NBA/NHL Finals, autoriser cote single
jusqu'à 2.50 SI :
- Sport NBA ou NHL
- proba_shrunk ≥ 0.40
- ≥ 2 sources sharps disponibles
- H2H récent favorable underdog (≥ 3 wins sur 5 H2H récents)
- Différentiel repos ≥ +2 jours en faveur de l'underdog
Le tier reste FLOOR (mise 2€) — c'est un PICK PRUDENT sur underdog cohérent,
pas un coup de poker.
```

**Proposition #2 — Élargir F1 combo en présence de boost bwin**
```diff
- Combiné 2 jambes : cote totale finale 1.60 – 2.20
+ Combiné 2 jambes : cote totale finale 1.60 – 2.50 SI boost bwin ≥ +20%
+ Combiné 2 jambes : cote totale finale 1.60 – 2.20 SINON
```
Justification : sur n=2 combos, les 2 avaient boost bwin et cote >2.20. Le
WIN (Ruud+Knicks 2.36) montre que le boost crée une vraie value au-delà
du seuil v3 strict. Le LOSS (Olympiakos 2.25) était dû à AB-1 (Ruud), pas
au seuil cote.

**Proposition #3 — Renforcer shrinkage anti-overconfidence**
```diff
- proba_shrunk = (n_eff × model_proba + 2 × book_proba) / (n_eff + 2)
+ proba_shrunk = (n_eff × model_proba + w_book × book_proba) / (n_eff + w_book)
+ où w_book = 2 si spread |model - book| ≤ 10 pts, 3 si > 10 pts, 4 si > 20 pts
```
Cela amortit automatiquement les estimations trop optimistes.
