  → 1/1.60 = 0.625 (implied book, NOT compté comme source quanti distincte
  — c'est notre `cote_ref` qui sert au book_proba)

**Sources qualitatives convergentes** :

  signal nuance — 5 sets = match dur)
- SportyTrader : "Ben Shelton to Win" (convergent)

  widen the gap as matches progress" (convergent)

**Signal négatif documenté (v4.8 Étape 1.5)** :

  sur 10 derniers**, R1 propre 6-3 6-3 7-6 vs Vukic
- Shelton 33-21 vs top-100 sur 52 sem (61.1%) — Collignon **mieux** que
  top-100 récemment

**Dédup éditeur** :

  distinctes après dédup
- Tennis Tonic + SportyTrader + LWOS → 3 quali convergentes

**F4** : ≥ 1 quanti + ≥ 3 convergentes ✓ après dédup
**F4-fresh v4.8** : sources Dimers/Stats Insider publiées 27/05 = post-R1
Shelton (R1 le 25/05) ✓
**AB-7 v4.8** : Shelton R1 6-3 6-3 6-4 (straight sets, ~2h estimé, pas de
drapeau physique CNN/officiel) → **pas de dérate** AB-7

**Calculs** :
```
cote_ref     = 1.60 (médiane consensus marché)
book_proba   = 1/1.60 = 0.625
model_proba  = MÉDIANE(0.64, 0.63, 0.625) = 0.630
n_eff        = 3 (Dimers, Stats Insider, sportsbook implied)
w_book       = 2 (FIXE v4.1)

EV           = 0.628 × 1.60 − 1 = 1.0048 − 1 = +0.5%
```

**Verdict technique** : EV +0.5% < +2% strict → **🟠 BORDERLINE** au mieux
mais avec **F3 durci v4.8** appliqué :
- EV ∈ [0%, +2%] (zone hors borderline standard) ET signal qualitatif
  négatif documenté (Collignon hot streak 70% win rate documenté) →
  **🔴 INSUFFISANT** (downgrade de 1 cran)

→ **REJET TOP 3 visible** (mais conservé pour trace).
