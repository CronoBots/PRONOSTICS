# 📊 Framework d'analyse — Pick safe du jour

> **Référentiel exhaustif des points à analyser pour chaque pari.**
>
> À utiliser à chaque pick quotidien. Objectif : couvrir TOUTES les
> catégories ci-dessous (pas obligatoirement 1 bullet par sous-point,
> mais s'assurer qu'aucune catégorie n'a été oubliée).
>
> Cible : **~50 points par pick**, structurés en sections nommées
> (`##🎯 Section` dans la liste `rationale`).

---

## 🧭 Méthode globale

1. **Sourcer largement** : minimum 3 sources web indépendantes
   (ESPN, official league, expert analysts type Lineups/Pickdawgz)
2. **Croiser les données** : refuser un pick si 2 sources contredisent
   sur un fait critique (ex: lineup, blessure)
3. **Estimer la probabilité honnêtement** : pas de biais "je veux que
   ça gagne". L'edge doit être réel ou il n'y a pas de pick
4. **Documenter les risques** : section "Risques" obligatoire avec
   3-5 scénarios de défaite réalistes
5. **Confirmer en cote ≥ 2.00** : sinon on n'a pas de value (ou on
   accepte avec preuve d'edge >= +10%)

---

## A. ##🎯 LE MATCH (contexte général)

- [ ] Sport + compétition (Premier League / NBA Playoffs / WTA 1000…)
- [ ] Format (1 match, série best-of-7, etc.)
- [ ] Date + heure de coup d'envoi (avec fuseau horaire)
- [ ] Stade / lieu + capacité
- [ ] Type d'opposition (rivalité, finale, derby)
- [ ] Enjeu (titre, qualification, descente, prestige)
- [ ] Diffusion (chaîne TV, audience attendue)

## B. ##📊 BILAN SAISON & CLASSEMENT

- [ ] Classement actuel des 2 équipes (ranking ligue/circuit)
- [ ] Bilan global (V-D-N, % victoires)
- [ ] Bilan à domicile vs à l'extérieur (statistique CLÉ)
- [ ] Bilan vs équipes du même niveau / supérieur / inférieur
- [ ] Différentiel de points/buts (run differential, point differential)
- [ ] ATS record (Against The Spread) — si parié au handicap
- [ ] Over/Under tendance historique
- [ ] Performances en playoffs / finales (si applicable)

## C. ##⭐ STARS & EFFECTIF DISPONIBLE

- [ ] Star principale équipe A : nom, position, KPIs récents
- [ ] Star principale équipe B : nom, position, KPIs récents
- [ ] Statut blessures (probable, doubtful, out, day-to-day)
- [ ] Lineup attendue (avec sources officielles ou rumeurs récentes)
- [ ] Joueurs absents (impact estimé)
- [ ] Joueurs suspendus (cartons, fights, etc.)
- [ ] Rotation prévue / minutes attendues
- [ ] Recrues récentes / trades / changements d'effectif

## D. ##💪 STATS AVANCÉES SPORT-SPÉCIFIQUES

### Football (soccer)
- [ ] xG (Expected Goals) attaque + défense
- [ ] Possession % moyenne
- [ ] Tirs cadrés / total
- [ ] Buts marqués / encaissés / clean sheets
- [ ] Set pieces conversion (corners, free kicks)
- [ ] PPDA (passes par action défensive) — pressing intensity
- [ ] Formation tactique + match-up positions

### Basketball (NBA, EuroLeague)
- [ ] Net Rating (ORtg - DRtg)
- [ ] Offensive Rating + Defensive Rating détaillés
- [ ] Pace (possessions/match)
- [ ] eFG% (Effective Field Goal)
- [ ] 3-point % + tentatives/match
- [ ] Free Throw % + tentatives/match
- [ ] Rebonds offensifs / défensifs
- [ ] Turnovers / Assists ratio
- [ ] Banc : minutes + net rating
- [ ] Match-up positions clés (qui défend qui)

### Tennis (ATP, WTA)
- [ ] Classement officiel + ELO rating (Tennis Abstract)
- [ ] Surface du match (clay, grass, hard, indoor)
- [ ] Win % sur cette surface (carrière + saison)
- [ ] Service : 1ère balle %, aces/double-fautes ratio
- [ ] Return : break points convertis / sauvés
- [ ] Service games won % vs Return games won %
- [ ] Durée moyenne des matchs (endurance proxy)
- [ ] Forme physique récente (blessures, retraits)

### Baseball (MLB)
- [ ] Lanceur partant : ERA global + ERA road/home séparés
- [ ] WHIP, BABIP, BB%, K%, HR/9 du lanceur
- [ ] Bullpen ERA récent (10 derniers jours)
- [ ] Lineup OPS contre lanceurs droitiers / gauchers
- [ ] Frappeurs en forme (xBA, hard hit %)
- [ ] Stade dimensions (park factors)
- [ ] Météo (vent direction, humidité)
- [ ] Probable starter J+1 (impact sur gestion bullpen)

### Hockey (NHL)
- [ ] Goal differential (5v5)
- [ ] Power play % + Penalty kill %
- [ ] Faceoff %
- [ ] Gardien : Save % récent + GAA
- [ ] Corsi/Fenwick (possession proxies)
- [ ] Back-to-back games
- [ ] Goalie rotation (qui démarre)

### NFL / NCAA Football
- [ ] Yards par match (offense + défense)
- [ ] Turnovers ratio
- [ ] 3rd down conversion % offense/défense
- [ ] Red zone TD %
- [ ] Time of possession
- [ ] Sacks / pressures
- [ ] Météo (vent affecte le jeu aérien)

## E. ##🤝 HEAD-TO-HEAD (H2H)

- [ ] H2H carrière complet entre les deux équipes/joueurs
- [ ] H2H sur les 5 dernières confrontations
- [ ] H2H sur cette surface/dans ce stade
- [ ] H2H quand équipe X reçoit ou visite
- [ ] H2H en playoffs / finales si applicable
- [ ] Patterns récurrents identifiables
- [ ] Dernière confrontation : score + analyse

## F. ##🔥 FORME RÉCENTE

- [ ] 5 derniers matchs : résultats + scores
- [ ] 10 derniers matchs : record W-L
- [ ] Performance contre opposition de niveau similaire
- [ ] Stats en hausse ou baisse (régression à la moyenne ?)
- [ ] Hot streak / cold streak en cours
- [ ] Comparaison forme actuelle vs moyenne saison

## G. ##🛌 FACTEURS PHYSIQUES

- [ ] Jours de repos depuis le dernier match
- [ ] Back-to-back games (NBA, NHL)
- [ ] Distance de voyage + fuseau horaire
- [ ] Altitude (Denver, La Paz, etc.)
- [ ] Fatigue cumulée (jeux consécutifs en road trip)
- [ ] Différentiel de repos entre les 2 équipes

## H. ##🧠 FACTEURS PSYCHOLOGIQUES

- [ ] Motivation / enjeu réel pour chaque équipe
- [ ] Pression médiatique (sponsoring, attentes)
- [ ] Revenge game / anniversary game
- [ ] Confiance actuelle (déclarations presse récentes)
- [ ] Historique mental dans les "big games"
- [ ] Coach pression (siège chaud, contrat)
- [ ] Public attendu (hostile, friendly, neutre)

## I. ##🌦️ CONDITIONS EXTERNES

- [ ] Météo : température, vent, pluie, humidité
- [ ] Heure du match (jour/soir, prime time ?)
- [ ] Arbitrage : historique de l'arbitre/umpire (homer ratio)
- [ ] Site : home, away, neutre
- [ ] Stade : capacité, ambiance attendue
- [ ] Conditions de pelouse / glace / parquet

## J. ##💰 MARCHÉ & COTES

- [ ] Cote moneyline (vainqueur) — bookmaker principal
- [ ] Cote spread / handicap si applicable
- [ ] Cote total over/under
- [ ] Line movement (cote d'ouverture vs cote actuelle)
- [ ] Money split (% du money sur chaque côté)
- [ ] Sharp money signals (line moves contre le public)
- [ ] Cote à plusieurs bookmakers (best line shopping)
- [ ] Cote boostée disponible (promo)

## K. ##🌐 CONSENSUS & MODÈLES TIERS

- [ ] FanGraphs / Tennis Abstract / 538 prediction
- [ ] ESPN BPI / Football Power Index
- [ ] DraftKings / FanDuel AI predictions
- [ ] Lineups.com expert grade
- [ ] Pickdawgz / Bleacher Nation expert picks
- [ ] Vegas Insider consensus
- [ ] Sentiment Twitter/X (sharp tipsters publics)

## L. ##🧮 NOTRE PROBABILITÉ & EDGE

- [ ] Notre estimation finale (chiffrée, justifiée)
- [ ] Probabilité implicite du bookmaker (1/cote)
- [ ] Edge = notre proba − proba bookmaker (en points)
- [ ] EV (Expected Value) = (proba × cote) − 1
- [ ] Comparaison edge vs minimum requis (5%)
- [ ] Variance attendue du sport (faible / forte)

## M. ##💵 GESTION DE LA MISE

- [ ] Mise recommandée (% bankroll)
- [ ] Calcul Kelly fraction (¼ Kelly recommandé)
- [ ] Gain potentiel net
- [ ] Perte potentielle
- [ ] Impact bankroll post-match (win / loss)
- [ ] Drawdown max acceptable atteint ?

## N. ##⚠️ RISQUES HONNÊTES

- [ ] Scénario où on perd (le plus probable)
- [ ] Variance du sport / "black swan" possibles
- [ ] Pourquoi le marché peut avoir raison contre nous
- [ ] Faiblesse de notre analyse identifiée
- [ ] Sources contradictoires (s'il y en a)
- [ ] Confidence rating final (Low / Medium / High)

## O. ##📚 SOURCES

- [ ] Minimum 3 URLs vérifiables
- [ ] 1 source officielle (ligue, fédération)
- [ ] 1 source analyste pro (Lineups, Pickdawgz, etc.)
- [ ] 1 source données chiffrées (FanGraphs, Basketball-Reference, etc.)
- [ ] Cote bookmaker source confirmée
- [ ] Date d'accès aux données (information fraîche < 24h)

---

## ✅ Check-list de fin

Avant de pousser un pick, vérifier que :

- [ ] **Cote ≥ 2.00** (ou edge documenté > +10%)
- [ ] **Toutes les sections A-N** ont au moins 1 bullet (sauf si vraiment NA)
- [ ] **Minimum 3 sources** dans la liste `sources`
- [ ] **Edge > +5%** entre notre proba et celle du bookmaker
- [ ] **EV > +5%** strictement positif
- [ ] **Rationale ~50 points** (45-60 acceptable)
- [ ] **Mise** cohérente avec Kelly fractionnel (¼ ou ½)
- [ ] **Section Risques** présente avec 3+ scénarios
- [ ] **Pas de promesse de gain** dans la formulation

## 🔄 Workflow quotidien

```
1. Recherche multi-sources (WebSearch + WebFetch)
   ↓
2. Listing des matchs candidats du jour (tous sports)
   ↓
3. Pour chaque candidat : remplir la checklist A-N
   ↓
4. Calcul edge et EV de chaque candidat
   ↓
5. Comparaison : ne garder QUE le pick avec
   - Cote ≥ 2.00
   - EV >= +5%
   - Edge >= +5 points
   - Confidence rating >= Medium
   ↓
6. Si 0 candidat qualifie → "Aucun value bet aujourd'hui"
   ↓
7. Si 1+ candidat → choisir celui avec edge × confidence maximum
   ↓
8. Rédaction du rationale exhaustif (sections + 45-60 bullets)
   ↓
9. Ajout à picks_data.py
   ↓
10. python scripts/build_history.py
    ↓
11. git commit + push → Pages redéploie
```
